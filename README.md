- Docker

## Getting Started

### Prerequisites

- Installed node modules:

```shell
yarn install
```

### Compilation

```shell
yarn compile
```

## Testing

```shell
yarn start-sandbox
yarn test
yarn stop-sandbox
```

## Deploy

```shell
yarn migrate
```
